# Database of British and Irish Hills

**Note**: This is a large file, and not our responsibility, so is not under version control.

## Directory structure
Put the DoBIH*.csv file in this directory (e.g. `app/skills/mwis-website/identify_forecast_area/resources/DoBIH-v18_4.csv`).

## Obtaining the data
Download the csv file from: https://www.hill-bagging.co.uk/dobih-downloads/hillcsv.zip
