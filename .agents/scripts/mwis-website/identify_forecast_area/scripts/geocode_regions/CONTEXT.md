Script that might be used occasionally to geocode the boundaries of each MWIS region based on a map. If you are using this again make sure to ask permission first. It may not be needed as there may be extra information you are not aware of.

The script was initially generated using the following prompt.

Using the MWIS regionCodes in 
../../reference/mwis-regions.csv
and 
assets/mwis-map.jpg
  which shows the UK overlaid with the 10 regions, geocode the boundaries of each MWIS region and put the result into a file you can access easily (JSON perhaps?). Put that file in the same directory as 
mwis-map.jpg
 . Keep a copy of the scripts you generate in scripts/geocode_regions/
